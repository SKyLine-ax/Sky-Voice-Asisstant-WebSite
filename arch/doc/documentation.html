Главная Документация Sky Voice Assistant

Голосовой Ассистент для управления компьютером

Голосовой Ассистент (ГА) — это программное решение, позволяющее управлять компьютером с помощью голосовых команд. ГА разработан для улучшения доступности компьютера и повышения эффективности работы пользователя.

Примечание: Данная документация описывает версию Sky Voice Assistant 0.0.1 (ВЕРСИЯ ЕЩЕ НЕ ВЫШЛА)

Основные возможности

Sky Voice Assistant предоставляет широкий набор возможностей для взаимодействия с компьютером:

  • Выполнение системных функций (регулировка громкости, смена раскладки)
  • Открытие ютуба,и самого браузера.
  • Открытие и закрытие программ
  • Треки в Spotify (Функция не работает без премиум Spotify)

Голосовые команды

Голосовой Ассистент поддерживает следующие типы голосовых команд:

Системные команды

Позволяют управлять функциями операционной системы.

Примеры: "Смени раскладку", "Звук на максимум", "Выключи звук"

Команды для приложений

Управляют конкретными приложениями и их функциями.

Примеры: "Открой браузер", "Закрой Дискорд", "Открой ютуб"

Управление системой

Голосовой Ассистент может управлять различными аспектами операционной системы:

Функция Примеры команд Описание
Работа, отдых "Перейди в режим отдыха", "Перейди в режим работы" Управление окнами и редактором кода
Настройки системы "Звук на половину", "Выключи звук" Изменение звука устройства
Языковая раскладка "Смени языковую раскладку", "Предыдущая языковая раскладка" Смена раскладки клавиатуры

Пример взаимодействия с системой

Пользователь: "Скай"
Ассистент: *звуковой сигнал активации*
Пользователь: "Открой браузер"
Ассистент: *открывает браузер (который установлен у вас по умолчанию)*
Пользователь: "Смени языковую раскладку"
Ассистент: *меняет раскладку на следующую в списке*
Пользователь: "Открой телеграмм"
Ассистент: *открывает Telegram*

Управление приложениями

Голосовой Ассистент предоставляет специальные команды для часто используемых приложений:

  • Браузер: открытие браузера по умолчанию, закрытие браузера, открытие YouTube
  • Spotify: воспроизведение, пауза трека
  • Discord: открытие Discord, закрытие
  • Telegram: открытие Telegram, закрытие

Настройка и установка

Системные требования

Для корректной работы Голосового Ассистента требуются следующие минимальные характеристики системы:

Компонент Минимальные требования Рекомендуемые требования
Операционная система Windows 10 Windows 11
Процессор Dual-core, 2.0 GHz Quad-core, 2.5 GHz+
Оперативная память 100 MB 1 GB или больше
Свободное место 200 MB 2 GB
Микрофон Встроенный или внешний микрофон Качественный направленный микрофон
Подключение к интернету Для онлайн-функций Стабильное широкополосное подключение
Важно: Голосовой Ассистент еще на стадии разработки. Системные требования пока что не точные. И могут быть изменены

Процесс установки

Установка Голосового Ассистента выполняется в несколько простых шагов:

  1. Скачайте установочный файл с официального сайта
  2. Запустите установщик и дождитесь завершения
  3. Укажите папку для установки
  4. Дождитесь завершения установки
  5. Все готово, наслаждайтесь голосовым управлением над своим компьютером
Важно: ГА еще на стадии разработки. Процесс установки может быть изменен

Устранение неполадок

Неполадок пока что не было обнаружено, если вы столкнулись с такими просьба связаться с нами.

Подсказка: Связь с нами вы можете найти на главной странице.

Для разработчиков

Данная секция предназначается для разработчиков которые хотят сделать своего ГА на основе Ская, ну либо же для тех кому интересно что было использовано.

Язык и библиотеки

Sky Voice Assistant написан на Python и использует следующие библиотеки:

Язык Зависимости Функция
Python simpleaudio Воспроизведение аудиофайлов
Python pyttsx3 Текст-в-голос (TTS) без интернета
Python pyaudio Работа с аудиопотоками
Python psutil Мониторинг системных ресурсов
Python pyautogui Автоматизация управления мышью и клавиатурой
Python yt-dlp Загрузка видео и аудио с YouTube и других сайтов
Python spotify Взаимодействие с API Spotify
Python comtypes Работа с COM-интерфейсами Windows
Python pycaw Управление громкостью в Windows
Python speechrecognition Онлайн-распознавание речи (Google, IBM и др.)
- Vosk Офлайн-распознавание речи
Подсказка: Да бы быстро установить все зависимости в терминале перейдите в папку с проектом а затем введите: pip install -r requirements.txt

API ключ Spotify

И так в Ассистенте используеться API ключи Spotify для воспроизведение треков пользователя. Вы можете либо вырезать этот функционал либл оставить. Что вам понадобиться:

  • CLIENT_ID
  • CLIENT_SECRET
  • REDIRECT_URI (В КОДЕ УКАЗАНА ССЫЛКА ЛУЧШЕ ЕЕ НЕ МЕНЯТЬ, СКОПИРОВАТЬ И ВСТАВИТЬ В ВАШЕ ПРИЛОЖЕНИЕ)
  • TOKEN_FILE (JSON файл где будет храниться универсальный токен пользователя, имееться по умолчанию)
Подсказка: Чтобы бы создать приложение Spotify вам нужно перейти на официальный сайт: https://developer.spotify.com
